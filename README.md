# Ford GoBike Data Analysis

## Overview

This project analyzes bike-sharing data from the Ford GoBike system. The objective is to understand user behavior, trip patterns, and key factors affecting bike-sharing usage.

## Business Problem

Bike-sharing companies need to understand how customers use their services in order to improve operational efficiency and customer satisfaction.

## Tools Used

* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Data Cleaning

* Removed missing values
* Converted date and time columns

### Time Analysis

* Daily trends
* Weekly trends

## Key Insights

* Subscribers generated the majority of trips.
* Peak demand occurred during commuting hours.
* Most trips were relatively short, indicating daily transportation use.
* User behavior varied significantly based on membership type.

## Business Recommendations

* Increase bike availability during peak commuting hours.
* Create targeted promotions for casual customers.
* Optimize station placement based on demand patterns.
